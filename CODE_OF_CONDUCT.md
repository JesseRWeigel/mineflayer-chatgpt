# Code of Conduct

## The Short Version

Be respectful, be constructive, be collaborative. We're here to build cool Minecraft bots together.

## Expected Behavior

- Be kind and constructive in code reviews, issues, and discussions
- Give and receive feedback gracefully
- Focus on the work — critique code, not people
- Help newcomers learn; everyone was a beginner once

## Unacceptable Behavior

- Personal attacks, insults, or harassment of any kind
- Spam, trolling, or deliberately disruptive behavior
- Publishing others' private information without permission

## Enforcement

The maintainer ([@JesseRWeigel](https://github.com/JesseRWeigel)) reserves the right to remove comments, close issues, or ban contributors who violate this code of conduct.

## Scope

This applies to all project spaces — issues, PRs, discussions, and any related communication channels.
